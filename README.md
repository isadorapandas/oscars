# The Oscars' Gems

Esta página foi feita com o objetivo de concentrar de forma visual e informativa alguns filmes cujas atrizes receberam indicações ao Oscar em suas respectivas temporadas.
O projeto foi proposto durante a Imersão Dev da Alura, que ocorreu em novembro de 2025. 
Foram utilizados os conceitos de HTML, CSS, JavaScript com o auxílio do Google Gemini.
